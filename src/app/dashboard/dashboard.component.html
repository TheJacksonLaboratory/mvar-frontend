Search for mouse variants

Model organisms are essential to understanding the biological and disease consequences of human genome variation. Bioinformatics resources that support meaningful comparisons of mouse and human genotype- to-phenotype data and knowledge are needed to support the translation from bench to bedside and back again.

MVAR Data Source

Currently the MVAR data consist of Mouse Variation data from the Sanger Institute (https://www.sanger.ac.uk/data/mouse-genomes-project/), which has been pre-processed and annotated (type of change, predicted functional consequence) using a custom pipeline. The database will be soon be updated with data from the Mouse Genome Informatics (Jax), dbSNP (NCBI), the Mouse Mutant Repository database (Jax) and the Diversity Outbred Database (Jax).

MVAR API details

The Mouse Variation Registry supports querying/registration using its API. The registry can be searched using Endpoints, which enable users to search the database by Variant, Strain, Gene and Transcript. The MVAR API is based on HTTP/HTTPS requests. Three type of requests are supported by this API: GET, POST and PUT. All requests can be sent through any browser using the corresponding URL in the address bar of the browser.
