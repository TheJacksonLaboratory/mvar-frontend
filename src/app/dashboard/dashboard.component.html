Mouse Mutant Resource Database

The Mouse Mutant Resource Database provides access to naturally occurring and induced variant data from laboratory mouse strains with Mendelian disease phenotypes, as well as phenotypically normal controls across more than 50 genetic backgrounds. The variant data are aggregated to facilitate rapid discovery of rare variants, powering the discovery of naturally occurring mutations that underlie (primarily) Mendelian phenotypes. Phenotypes are annotated using high level Mammalian Phenotype Ontology (http://www.informatics.jax.org/vocab/mp_ontology) terms, however for many of the strains, additional phenotype information is available through the Mouse Genome Database by searching strains by the strain ID (typically the JAX strain ID) or by searching genes by allele symbol or name (www.informatics.jax.org). Many of the phenotypes have been genetically mapped and chromosomal linkage data are provided. Where linkage data were not available at the time of sequencing, sibling control data were included. MMRdb uses inheritance, chromosomal linkage (where available), and variant occurrence data to nominate variants for causation. Aggregated, filtered variant calls (across many samples) or per sample VCF files can be exported. Tools for uploading variant call sets for annotation are underway. In the meantime, user variant calls can be integrated for annotation by contacting us directly. For more information about MMRdb and MMRdb data, see the About page.
