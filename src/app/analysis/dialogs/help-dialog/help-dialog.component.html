SNPs/INELs

Variant calls from reference (GRCm38) based alignment of Illumina short read data (exome and whole genome), and generated from a standard GATK pipeline with modification for laboratory mouse genomes (Fairfield et al., 2015).

Position – the physical position of the variant in the mouse genome reference build GRCm38

Type – the type of variant, SNP: single nucleotide polymorphism, DEL: deletion, INS: insertion

Impact – predicted impact annotation, e.g. from SNPEff, Putative_impact: A simple estimation of putative impact / deleteriousness : (HIGH, MODERATE, LOW, MODIFIER), reference http://snpeff.sourceforge.net/SnpEff_manual.html for details

Functional Class – predicted variant function according to position and base change, e.g. from SNPEff, Functional class (NONE, SILENT, MISSENSE, NONSENSE), reference http://snpeff.sourceforge.net/SnpEff_manual.html for details

Occurrence % - the frequency of the variant within the MMRdb variant database by variant position

Mutation Candidate – A variant is a mutation candidate if its occurrence is low (0-1% frequency where variants found in a single sample are ~0%), its genotype is supported by the phenotype of the sample and the inheritance of the phenotype (e.g. for autosomal recessive phenotypes, an affected individual must be homozygous), and if linkage information is available, the variant must be linked)

Sample ID – The ID assigned to each sample when it is added to MMRdb. Use this I.D. to search for additional sample information under ‘Samples’

Samples

Sample ID – The ID assigned to each sample when it is added to MMRdb. Use this I.D. to search for additional sample information under ‘Samples’

Sample Name – The name of the sample provided by the owner of the sample / data

JAX Registry ID – This is the JR or JAX strain ID which can be used to search the Jackson Laboratory Strain database or the Mouse Genome Database for additional strain information. Link out to the JAX Strain datasheet and ordering information

LIMS sample ID – This is the sample ID that can be used to track sample preparation and sequencing information if the data were generated by Genome Technologies at The Jackson Laboratory

Chromosome Linkage – If mapping data are available for the phenotype, this is the chromosome containing the SNPs with the most significant LOD scores

Genotype – this is inferred (if possible) from inheritance and phenotypic status of the sample

Study – This is the identifier for the study from which the data were generated. WES: whole exome sequencing (paired end is default), WGS: whole genome sequencing (paired end is default), MP: mate pair or long jump library, SE: single end

Platform – relevant to whole exome sequencing, this is the version of the exome capture reagents

Likely pathogenic – A variant is a likely pathogenic (y) if its occurrence is low (0-1% frequency where variants found in a single sample are ~0%), its genotype is supported by the phenotype of the sample and the inheritance of the phenotype (e.g. for autosomal recessive phenotypes, an affected individual must be homozygous), and if linkage information is available, the variant must be linked)

Confirmed mutations – This is reserved for those mutation candidate / likely pathogenic variants that have been confirmed through linkage analysis and / or PCR validation to be the putative mutation causing the phenotype that is annotated for a given sample. This field is user provided and is typically supported by published data
